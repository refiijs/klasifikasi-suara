nama file dan tujuan 
	
1. baca_dataspeech : itu fungsi buat deteksi suara yang dipake di main porgram

2. dataset.xlx : buat nyimpen dataset dari file wav, yang isi kolomnya filename (alamata dan namafile) dan class (dog or cat)

3. forXLX : itu buat generate dataset.xlx supaya ga namain satu satu

4. ekstraksi_mfcc : itu ekstraksi mfcc sesuai namanya dengan parameter y_speech, Fs, n_coeffs, L, S, view_mfcc (0 atau 1 kalau mau lihat mfccnya)) ini juga fungsi sama seperti baca_dataspeech

5. main.mlx: ini program utama berupa livescript

6. model_hmm.mat : ini hasil model yang sudah ditraining menggunakan hmm dan disave dengan format mat

7. main.mlapp: ini design app dari main program supaya ada guinya
